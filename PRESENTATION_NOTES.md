## Responsive Mobil Magaza - Sunum Notlari

### 1. Proje Amaci
- Responsive Mobil Magaza, mobil-first yaklasimla tasarlanmis bir e-ticaret arayuzudur.
- Hedef, urun listeleme, urun detay inceleme ve sepet deneyimini tek bir modern arayuzde sunmaktir.

### 2. Kullanilan Teknolojiler
- HTML5
- CSS3
- Tailwind CSS
- Vanilla JavaScript
- Vite

### 3. One Cikan Teknik Noktalar
- Mobil-first tasarim anlayisi ile gelistirildi.
- CSS Grid ve Flexbox birlikte kullanilarak telefon, tablet ve masaustu ekranlara uyum saglandi.
- Semantik HTML etiketleri tercih edilerek SEO ve Lighthouse uyumu desteklendi.
- Butonlar, ARIA etiketleri ve klavye odak durumlari ile erisilebilirlik guclendirildi.

### 4. Uygulama Ozellikleri
- Urun listesi ekrani
- Tiklandiginda guncellenen urun detay alani
- Sepete urun ekleme ve cikarma
- Toplam fiyat hesaplama
- Kategori bazli filtreleme

### 5. Performans ve Lighthouse Acisindan Yapilanlar
- Gorseller `loading="lazy"` ile optimize edildi.
- Meta description eklendi.
- Fazla kutuphane kullanilmadi, sade JavaScript tercih edildi.
- Renk kontrasti ve buton etiketleri duzenlendi.

### 6. Teslim Ciktisi
- Calisan responsive site
- Dokumante edilmis README
- Sunum icin hazir teknik ozet

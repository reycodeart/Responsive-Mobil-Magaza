# Responsive Mobil Magaza

Mobil oncelikli tasarlanmis tek sayfa magaz a uygulamasi.

## Ozellikler

- Responsive mobil magaza arayuzu
- Kategori filtreleme
- Urun kartlari ve kampanya alani
- Sabit alt gezinme
- Hafif ve performans odakli Vite + React yapisi

## Komutlar

- `npm install`
- `npm run dev`
- `npm run build`
